Where is my data stored?
Your workout data is saved only on this device, in your browser's local storage. There is no account and no cloud backup — if you clear your browser data, history, or site cache, your logs will be permanently deleted.

Use Export Data regularly to save a backup file. The export is a .json file containing all your logs, custom exercises, settings, and favourites. You can re-import it anytime using Import Data — it only accepts .json files exported from this app, and merges them with any existing data (nothing is overwritten).
